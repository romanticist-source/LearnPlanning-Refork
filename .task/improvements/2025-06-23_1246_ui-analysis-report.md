## 技術的特徴

### 使用技術スタック
- Next.js 15 + React 19
- TypeScript (strict mode)
- Tailwind CSS + shadcn/ui
- Radix UI primitives
- date-fns (日本語対応)
- NextAuth.js v5 beta

### コード品質
- **高品質**: 一貫したコンポーネント設計
- **アクセシビリティ**: Radix UI使用で配慮済み
- **レスポンシブ**: Tailwind CSS適切使用
- **型安全性**: TypeScript完全活用
